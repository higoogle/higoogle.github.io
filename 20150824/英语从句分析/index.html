英语从句分析

  • 英语 English

从句分析

翻译英文文档的时候，看到语句长句，挺有意思的，下面分析一下，怎么拆句，断句才合理，怎么翻译才更加的到位

首先，这个从句是状语从句+名词性从句+标语从句，那么下面先来拆句：

  • 第一遍：
  1. When a virtual method is called on a reference
  2. the actual type of the object to which the reference refers is used to determine which method implementation should be used
  • 第二遍:
  1. the actual type of the object to which the reference refers is used to determine
  2. which method implementation should be used
  • 第三遍拆句:
  1. the actual type is used to determine
  2. of the object to which the reference refers

句子拆完了，下面来开始翻译：

  1. 实际类型是用来被用来确定的
  2. 实际类型是用来确定哪个方法的实现会被使用
  3. 那些引用引用的对象的实际类型是用来确定哪个方法的实现会被使用
  4. 当一个虚方法被引用调用的时候，那些引用引用的对象的实际类型是用来确定哪个方法的实现会被使用。
  5. 当一个虚方法被引用调用的时候，类中的哪个方法的实现会被调用，这是由那些引用引用的对象的实际类型来决定的。
