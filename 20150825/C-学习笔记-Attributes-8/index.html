Attribute

在程序中，attribute是一个定义的标记，它可以给运行时传递各类元素的信息，比如：类，方法，结构体，emue等。

你可以通过attribute添加一个说明信息给程序.一个说明标记是通过[]来使用的。
attribute被用来添加元数据，比如complier isntruction,comments,descritions,methods,class等等。
.Net提供两种类型的attrubutes:

  1. 预定义的attribute
  2. 自定义构建的attribute

废弃的（Obsolete）

它标记个程序实体不会被使用。它能够让你告诉编译器去丢弃特定的目标元素。比如，当一个类的新方法正在被使用的时候，如果你仍然想保持使用类中的旧的的方法，通过展示一个信息给新方法，你可以 标记它为’obsolete’，让旧方法被使用

parameter message 是一个描述一个为什么它被丢弃，并且什么应该被使用。
parameter iserror 是一个Boolean的值，如果是true，那么编译器就应该认为程序的是错误的。默认的值是false（编译器会报一个警告)

当你尝试编译程序的时候，编译器会给你一个错误：
Don’t use OldMethod, use NewMethod instead

创建自定义的Attributes

创建制定的Attribute需要如下4步骤:

  1. 定义一个自定义atttribute
  2. 构造自定义attribute
  3. 应用自定义attribte到一个程序元素目标上
  4. 通过放射来访问

最后一步需要写一个简单程序来读元数据来发现各种符号（notations）
元数据是一种数据，信息，被用来描述其他的数据的。程序应该通过使用放射来在运行时访问属性。
