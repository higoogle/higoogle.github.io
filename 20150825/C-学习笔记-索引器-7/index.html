C#学习笔记-索引器(7)

  • c#

索引(indexer)

索引器容许一个对象想数组一样被索引访问。当给一个类定义索引的时候，这个类的行为就想个虚拟的数组一样。你可以通过数组的访问符[]来访问该对象。

语法

使用indexer

索引器的定义，某种程度上像一个属性。和属性一样，你可以使用get,set来给索引器定义。然而，属性返回或者设置特定的data member，然而索引返回或者设置一个来之 instance object 的值。从另一方面说，索引器将instance object data 打碎成一个一个小的值和对应的索引。

定义一个属性需要提供属性的名称，indexer不要提供名称，但是需要是使用this关键词，这个this关键词引用的是实例对象。

重载索引器

索引器可以重载。索引器也可以定义多个参数并且每个参数的类型都可以是不同的类型的。不一定非要是int类型的索引器。
