C#学习笔记(1)

  • c#
  1. C#中所有的值类型都继承于System.ValueType,然而System.ValueType又继承于System.Object.
  2. System.Object是所有的类的基类
  3. int = Int32
  4. bool = Sytem.Bool 以此类推。。。
  5. 变量首字母小写，方法首字母大写
  6. int x = 0; 等价于 int x = new int(); //不同于Java
  7. F12查看类的定义
  8. 就拿int类型说明，int是struct类型，它本身有些方法，可以通过.来调用
  9. 枚举: enum Week{Mon,Thu,Wen,Tus,Fri,Sat,Sun}; 打印的结果是Mon,而非类是OC,Java的0，1，2等,不过可以强制转化为int类型，这个机制是通用的。
  10. CSharp 3 推出var.任何类型都可以使用var来申明，比如:var z = 100;
  1. CSharp3中内置类类型：Object,String,Dynamic.
  2. 关于类型大小的写：string username = "zhanshan"; == Stirng username = "zhanshan" ==System.String username = "zhanshan";同样的Object = object = System.Object`
  3. Object常用方法: GetType(),ToString()
  4. string可以使用 += 来处理。 string a = "aa"; a+= "bb";
  5. string a = "abcd"; a[0] ===> 'a'
  6. string a ="E:\\DEV" == string a = @"E:\DEV"
  7. 如果需要频繁的操作字符串，不推荐使用string,而要使用StringBuilder
  8. 因为频繁操作字符串，每次操作都会分配一次内存空间，而使用StringBuilder仅为一次
